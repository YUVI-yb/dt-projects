# Finance Assignment

This project is a structured financial reasoning exercise designed to demonstrate clarity in analysis, decision-making, and interpretation of financial scenarios.

---

📌 Objective

The goal of this assignment is not just to compute answers, but to demonstrate:

- Structured financial thinking
- Logical interpretation of given constraints
- Clear explanation of reasoning steps
- Ability to translate financial scenarios into decisions

---

🧠 Approach

Instead of focusing only on final outputs, the emphasis is on:

- Breaking down the problem into interpretable parts
- Identifying relevant financial logic
- Ensuring reasoning is traceable and explainable
- Avoiding assumptions without justification

---

📂 Contents

- Case analysis / problem breakdown
- Step-by-step reasoning
- Final conclusions based on structured interpretation

---

📎 How to Review

1. Open the assignment file(s) in this folder
2. Read the reasoning in sequence
3. Focus on the logic progression, not just final values

No setup or tools are required.

---

## 👤 Note

This assignment focuses on analytical thinking rather than computation-heavy financial modelling.
